# Eivind Gamst - rano-oss.github.io

Welcome to my personal blog and open-source hub.

## Blog
You can find my blog at: [https://rano-oss.github.io](https://rano-oss.github.io)

## About
This site is built using [Zola](https://www.getzola.org/), a fast static site generator written in Rust, and uses the [Kita](https://github.com/st1020/kita) theme.

## Development

### Prerequisites
- [Zola](https://www.getzola.org/documentation/getting-started/installation/)

### Local Setup
1. Clone the repository with submodules:
   ```bash
   git clone --recursive https://github.com/rano-oss/rano-oss.github.io.git
   ```
2. Serve the site locally:
   ```bash
   zola serve
   ```
3. Open `http://127.0.0.1:1111` in your browser.
